#Function Hall Website

## Project Overview

This project is a responsive Function Hall Business Website developed using HTML, CSS and JavaScript.

## Features

- Responsive Design
- Home Page
- About Section
- Services Section
- Facilities Section
- Gallery
- Packages
- Booking Form
- Smooth Scrolling
- Form Validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download project files.
2. Open folder in VS Code.
3. Open index.html using Live Server.
4. Website will run in browser.
